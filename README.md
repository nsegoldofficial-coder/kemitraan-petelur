# Kemitraan Petelur - Landing Page

Project ini berisi landing page untuk program kemitraan petelur.

## Struktur
- `mitra2/`: Landing page untuk Mitra 2
- `mitra3/`: Landing page untuk Mitra 3

## Cara Menggunakan
Buka file `index.html` di browser untuk melihat hasilnya.

## Kontak
Edit file HTML secara manual untuk memperbarui informasi kontak.
